# Sales & Revenue Analysis Dashboard

## Project Overview

This dashboard analyzes sales and revenue data using Python and Streamlit.

Features:

- KPI Tracking
- Revenue trends
- Top products
- Region analysis
- Interactive filters

---

## Technologies

- Python
- Pandas
- Plotly
- Streamlit

---

## Dataset Columns

Date

Product

Category

Region

Quantity

Sales

Revenue

---

## Dashboard KPIs

Total Sales

Total Revenue

Revenue Trends

Top Products

Region Performance

---

## Run Project

Install:

```bash
pip install -r requirements.txt
```

Run:

```bash
streamlit run app.py
```
